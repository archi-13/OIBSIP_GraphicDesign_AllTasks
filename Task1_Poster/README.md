# Task 1 — Poster & Flier Design

## Objective
Design a visually appealing digital poster for a 
public marketing campaign.

## Tools Used
- Figma

## Steps Performed
1. Selected marathon as the poster theme
2. Set up canvas in Figma
3. Added hero background image
4. Designed layout with event details, race categories,
   QR code and sponsor section
5. Exported as PNG and PDF

## Outcome
A professional event poster for Marathon 26 — Run for 
a Cause featuring date, location, race categories 
(5K, 7K, 10K), QR registration and sponsor logos.
