# Task 2 — Logo Design

**Intern:** Archi Lad  
**Domain:** Graphic Design  
**Tool Used:** Figma

---

## Objective
Design a minimal brand logo that represents a company's
identity and values following the KISS principle.

## Steps Performed
1. Decided on brand concept — StrideCo fitness brand
2. Designed runner icon using vector shapes in Figma
3. Selected Urban Stride color palette
4. Chose Nunito rounded typography to match icon style
5. Exported as PNG and PDF

## Outcome
A clean minimal logo for StrideCo featuring a custom
runner icon, terracotta and navy palette, and rounded
typography.
